# BindCraft

**BindCraft** is a client-side Fabric mod for Minecraft `1.21.11` that expands the vanilla keybinding system with customizable action sequences, profiles, and advanced key combinations.

Instead of binding a single key to a single action, BindCraft allows players to create intelligent bindings that execute multiple client-side actions in order.

## 🚀 Features

- **Multiple independent profiles** — Manage different setups easily.
- **Profile import and export** — Share your bindings with others.
- **Folder organization** — Keep your bindings tidy and structured.
- **Advanced key combinations** — Go beyond standard vanilla limits.
- **Sequential action execution** — Execute multiple commands in perfect order.
- **Individual toggles** — Enable or disable specific folders and bindings at will.
- **Vanilla-style UI** — Feels right at home in the Minecraft menus.
- **Lightweight & Client-side** — No server-side installation required.
- **JSON-based configurations** — Easy to read and manually edit if needed.
